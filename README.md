## 📌 About The Project
This repository contains a comprehensive collection of SQL Server queries, database schemas, and Entity-Relationship Diagrams (ERDs) developed during the **C15** track. It serves as a practical demonstration of database design principles, T-SQL scripting, and data manipulation techniques.

## 📂 Project Structure
The repository is organized into the following directories to keep the workflow clean and professional:

* **`/ERD&Schema`**: Contains the visual representations of the database architecture, including Entity-Relationship Diagrams (ERDs) and structural blueprints.
* **`/SQL-Queries`**: The core directory containing all `.sql` files. It includes scripts for:
  * Creating databases and tables (DDL).
  * Inserting, updating, and deleting records (DML).
  * Complex queries using Joins, Subqueries, and Aggregate functions.
  * Views, Stored Procedures, and Triggers (if applicable).

## 🚀 Built With
* **Microsoft SQL Server** (RDBMS)
* **T-SQL** (Transact-SQL)
* **SQL Server Management Studio (SSMS)**
